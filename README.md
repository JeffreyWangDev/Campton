# Campton
> Website developed for the Acton Lions Club to assist in the annual ski and skate sale.
# Getting Started
## Requirements
- Python 3.9+
- PIP
- Docker (Optional)
## Installation
```git clone https://github.com/JeffreyWangDev/Campton.git
cd Campton
py -m venv env
.\env\Scripts\activate
py -m pip install -r requirements.txt
```
